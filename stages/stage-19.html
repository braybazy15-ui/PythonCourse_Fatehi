📦 معرفی سه ساختار داده

تا حالا با لیست آشنا شدی. پایتون سه ساختار داده مهم دیگر هم دارد:

📦 تاپل (Tuple)

()

غیرقابل تغییر - مثل لیست اما نمی‌توانی آیتم‌هایش را تغییر دهی. با پرانتز ساخته می‌شود.

🔑 دیکشنری (Dict)

{}

کلید-مقدار - مثل دفترچه تلفن! هر مقدار با یک کلید مشخص می‌شود. با آکولاد ساخته می‌شود.

🎯 مجموعه (Set)

set()

بدون تکرار - مثل لیست اما آیتم‌های تکراری را حذف می‌کند. ترتیب ندارد.

📦 تاپل (Tuple) - لیست غیرقابل تغییر

تاپل دقیقاً شبیه لیست است، با یک تفاوت بزرگ: نمی‌توانی آیتم‌هایش را تغییر دهی، اضافه کنی یا حذف کنی! از تاپل برای داده‌هایی استفاده می‌کنی که نباید تغییر کنند (مثل روزهای هفته، مختصات جغرافیایی).

# ساخت تاپل با پرانتز days = ("شنبه", "یکشنبه", "دوشنبه") print(days[0]) # این خطا می‌دهد! # days[0] = "جمعه" # TypeError!
شنبه

🔑 دیکشنری (Dictionary) - دفترچه تلفن پایتون

دیکشنری مثل یک دفترچه تلفن است! هر کلید (Key) به یک مقدار (Value) اشاره می‌کند. به جای ایندکس عددی، از کلیدهای معنادار استفاده می‌کنی.

# ساخت دیکشنری با آکولاد student = { "name": "Ali", "age": 20, "grade": "A" } print(student["name"]) print(student["age"]) # اضافه کردن کلید جدید student["city"] = "Tehran"
Ali 20

🎯 مجموعه (Set) - بدون عضو تکراری

مجموعه مثل لیست است اما عضو تکراری نمی‌پذیرد و ترتیب خاصی ندارد. برای حذف موارد تکراری از یک لیست عالی است!
