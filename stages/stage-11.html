🔢 انواع داده (Data Types) در پایتون

آشنایی کامل با انواع مختلف داده‌ها و کاربرد هر کدام

👋 سلام، دانشجو جان! امروز با انواع داده‌ها آشنا می‌شیم 🔢

🧪 نوع داده (Data Type) یعنی چه؟

تا حالا با متن (رشته) و اعداد کار کردیم. اما آیا این دو تنها چیزهایی هستند که پایتون می‌تواند ذخیره کند؟ قطعاً نه! پایتون انواع مختلفی از داده‌ها را می‌شناسد و با هر کدام رفتار متفاوتی دارد.

نوع داده (Data Type) یعنی دسته‌بندی اطلاعات. اینکه یک داده از چه جنسی است: عدد است؟ متن است؟ درست یا غلط است؟ هر نوع داده، قوانین و کاربردهای خاص خودش را دارد.

🔤🔢✅

مثال واقعی: مثل دسته‌بندی وسایل!

تصور کن توی آشپزخانه، مواد غذایی را دسته‌بندی می‌کنی: ادویه‌ها یک نوع هستند، لبنیات نوع دیگر، سبزیجات نوع سوم. هر کدام ویژگی‌های خاص خودشان را دارند و نمی‌شود با همه‌شان یک‌جور رفتار کرد. نوع داده هم همین است!

📋 انواع داده اصلی در پایتون

پایتون چندین نوع داده اصلی دارد. بیا با مهم‌ترین‌هایشان آشنا شویم:

📝

رشته (String)

str

متن، کلمات، جملات. داخل " " یا ' ' قرار می‌گیرد. مثل: "Hello"

🔢

عدد صحیح (Integer)

int

اعداد بدون اعشار، مثبت یا منفی. مثل: 42, -10, 0

📏

عدد اعشاری (Float)

float

اعداد با ممیز اعشاری. مثل: 3.14, -0.5, 2.0

✅❌

بولین (Boolean)

bool

فقط دو مقدار: True (درست) یا False (نادرست)

📋

لیست (List)

list

مجموعه مرتب از آیتم‌ها. مثل: [1, 2, 3] یا ["a", "b"]

📦

تاپل (Tuple)

tuple

شبیه لیست ولی غیرقابل تغییر. مثل: (1, 2, 3)

🔍 چطور نوع یک داده را بفهمیم؟

پایتون یک تابع جادویی به نام type() دارد که به تو می‌گوید هر متغیر از چه نوعی است. مثل یک دستگاه تشخیص جنس!

# تشخیص نوع داده با type() name = "Ali" print(type(name)) # خروجی: age = 25 print(type(age)) # خروجی: pi = 3.14 print(type(pi)) # خروجی: is_happy = True print(type(is_happy)) # خروجی:

📝 نوع رشته (String - str)

رشته یعنی متن. هر چیزی که داخل کوتیشن (" " یا ' ') قرار بگیرد، رشته محسوب می‌شود. حتی اعداد هم اگر داخل کوتیشن باشند، رشته هستند!

text1 = "Hello World" # رشته text2 = 'Python' # رشته text3 = "123" # رشته (نه عدد!) text4 = "" # رشته خالی

ویژگی‌های رشته:

  • با + می‌توانی رشته‌ها را به هم بچسبانی: "Hello" + " " + "World"
  • با * می‌توانی رشته را تکرار کنی: "Ha" * 3 ← "HaHaHa"
  • می‌توانی طول رشته را با len() بگیری

🔢 نوع عدد صحیح (Integer - int)

عدد صحیح یعنی اعداد بدون ممیز و اعشار. می‌توانند مثبت، منفی یا صفر باشند.

a = 42 # عدد صحیح مثبت b = -10 # عدد صحیح منفی c = 0 # صفر هم عدد صحیح است d = 1000000 # عدد بزرگ - می‌تونی با _ جدا کنی: 1_000_000

📏 نوع عدد اعشاری (Float - float)

عدد اعشاری یعنی اعداد با ممیز اعشار. هر عددی که نقطه (.) داشته باشد، float است.

pi = 3.14159 # عدد اعشاری half = 0.5 # صفر و نیم negative = -2.7 # اعشاری منفی also_float = 5.0 # این هم float است، نه int!

✅ نوع بولین (Boolean - bool)

بولین ساده‌ترین نوع داده است. فقط دو مقدار دارد: True (درست/بله) یا False (نادرست/خیر). خیلی شبیه کلید چراغ برق: یا روشن است یا خاموش!

is_student = True # بله، دانشجو هست is_raining = False # خیر، باران نمی‌آید has_license = True # بله، گواهینامه دارد

نکته مهم: True و False حتماً باید با حرف بزرگ شروع شوند!

نامکلمه کلیدیمثالکاربرد

🧠 نکته طلایی: پایتون یک زبان Dynamic Typed است! یعنی لازم نیست نوع متغیر را از قبل مشخص کنی. پایتون خودش تشخیص می‌دهد که چه نوع داده‌ای در متغیر ریخته شده است. یک متغیر حتی می‌تواند در طول برنامه نوعش عوض شود!

🧪 تمرین‌های عملی انواع داده

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب را چک کن!

۱
نوع داده‌های زیر را با type() تشخیص بده: 42, "42", 3.14, True
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
print(type(42)) # int print(type("42")) # str print(type(3.14)) # float print(type(True)) # bool
۲
تفاوت "123" و 123 چیست؟ با type() نشان بده
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a = "123" # رشته - نمی‌توان جمع کرد b = 123 # عدد - می‌توان جمع کرد print(type(a)) # str print(type(b)) # int
۳
یک متغیر بولین برای وضعیت آنلاین بودن کاربر بساز
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
is_online = True print(f"وضعیت آنلاین: {is_online}") # خروجی: وضعیت آنلاین: True
۴
چطور بفهمی 5.0 از نوع float است یا int؟
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
x = 5.0 print(type(x)) # float # هر عددی که نقطه داشته باشد float است # حتی اگر مقدارش صحیح باشد!
۵
با اعداد 10 و 3 یک float و یک int جمع کن و نوع نتیجه را ببین
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a = 10 # int b = 3.5 # float c = a + b # 13.5 print(type(c)) # float # جمع int و float = float
۶
یک رشته را با * تکرار کن و طولش را با len() بگیر
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
text = "Hi" * 4 print(text) # HiHiHiHi print(len(text)) # 8 print(type(text)) # str
