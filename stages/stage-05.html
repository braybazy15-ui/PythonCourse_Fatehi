معرفی، نصب و راه‌اندازی Visual Studio Code

👋 سلام، دانشجو جان! امروز VS Code رو نصب می‌کنیم 💻

💻 VS Code چیست؟

Visual Studio Code (یا به اختصار VS Code) یه ویرایشگر کد رایگان، قدرتمند و محبوب است که توسط شرکت مایکروسافت ساخته شده. VS Code مثل یه دفترچه پیشرفته برای نوشتن کد می‌مونه که کلی امکانات داره تا کار برنامه‌نویسی رو برات راحت‌تر کنه.

چرا VS Code رو انتخاب می‌کنیم؟ چون:

🆓

کاملاً رایگان

هیچ هزینه‌ای نداره و برای همیشه رایگانه

🪟🍎🐧

چندسکویی

روی ویندوز، مک و لینوکس کار می‌کنه

🧩

اکستنشن‌ها

هزاران افزونه برای هر زبان برنامه‌نویسی

🎨

رنگ‌آمیزی کد

کدها رو رنگی نشون میده تا خواناتر بشن

💡

پیشنهاد هوشمند

موقع نوشتن کد بهت پیشنهاد میده (IntelliSense)

🐞

دیباگر داخلی

خطاهای کد رو پیدا و رفع می‌کنه

📌 تفاوت VS Code با پایتون:
• پایتون = زبانی که باهاش حرف می‌زنی و کد رو اجرا می‌کنه
• VS Code = دفترچه‌ای که توش کد رو می‌نویسی
پایتون مثل موتور ماشینه، VS Code مثل فرمان و داشبورد ماشین!

📥 دانلود و نصب VS Code

برای دانلود VS Code، باید بری به سایت رسمی:

🌐 آدرس سایت: https://code.visualstudio.com/
وارد سایت بشو. سایت به طور خودکار سیستم‌عامل تو رو تشخیص میده و دکمه دانلود مناسب رو نشون میده. روی دکمه "Download" کلیک کن.

🪟 نصب روی ویندوز

  1. فایل .exe دانلود شده رو اجرا کن.
  2. تیک «I accept the agreement» رو بزن و Next رو بزن.
  3. مسیر نصب رو تغییر نده (پیش‌فرض خوبه) و Next رو بزن.
  4. تیک گزینه «Add to PATH» رو حتماً بزن! (مثل پایتون)
  5. تیک «Create a desktop shortcut» رو بزن تا آیکون روی دسکتاپ ساخته بشه.
  6. Install رو بزن و صبر کن تا نصب تموم بشه.

🍎 نصب روی مک

  1. فایل .zip دانلود شده رو باز کن.
  2. فایل Visual Studio Code.app رو بکش به پوشه Applications.
  3. از Launchpad می‌تونی VS Code رو باز کنی.

🐧 نصب روی لینوکس (اوبونتو)

  1. از سایت، نسخه .deb رو دانلود کن.
  2. توی Terminal دستور زیر رو بزن: sudo dpkg -i code*.deb
  3. اگه خطای dependency گرفتی، بزن: sudo apt --fix-broken install

🧩 اکستنشن (Extension) چیست؟

اکستنشن یعنی افزونه یا پلاگین. یه تکه برنامه کوچک که به VS Code اضافه می‌کنی تا قابلیت‌های جدیدی بهش بده. درست مثل اپ‌هایی که روی گوشی نصب می‌کنی! هر زبان برنامه‌نویسی اکستنشن مخصوص خودش رو داره.

برای پایتون، بهترین و ضروری‌ترین اکستنشن، اکستنشن رسمی پایتون از مایکروسافت است که کلی قابلیت اضافه می‌کنه:

  • 🎨 رنگ‌آمیزی کد پایتون
  • 💡 پیشنهاد هوشمند کد (Auto-complete)
  • 🐞 دیباگ کردن کد
  • ▶️ اجرای کد با یک کلیک
  • ⚠️ نمایش خطاها و هشدارها
  • 📦 مدیریت محیط‌های مجازی

🔌 نصب اکستنشن پایتون

🐍

Python - by Microsoft

اکستنشن رسمی پایتون - ضروری برای هر برنامه‌نویس پایتون

  1. VS Code رو باز کن.
  2. از نوار سمت چپ، روی آیکون Extensions کلیک کن (یا کلیدهای Ctrl+Shift+X رو بزن).
  3. توی کادر جستجو بنویس: Python
  4. اولین نتیجه که نوشته "Python by Microsoft" با علامت تیک آبی رو انتخاب کن.
  5. روی دکمه "Install" کلیک کن و صبر کن تا نصب بشه.
  6. بعد از نصب، VS Code رو یکبار ببند و دوباره باز کن.
