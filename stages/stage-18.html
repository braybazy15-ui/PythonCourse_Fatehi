📋 لیست (List) چیست؟

تا حالا هر متغیر فقط یک مقدار نگه می‌داشت. اما فرض کن می‌خواهی نمرات ۳۰ دانش‌آموز را ذخیره کنی. آیا باید ۳۰ تا متغیر بسازی؟ قطعاً نه! اینجاست که لیست (List) به کمک می‌آید.

لیست مثل یک جعبه ابزار یا کیف است که می‌توانی هر تعداد چیزی داخلش بگذاری. هر آیتم در لیست، یک خانه دارد با یک شماره (index) که از صفر شروع می‌شود.

🧠 مثال واقعی: تصور کن یک قطار با ۵ واگن داری. واگن اول شماره ۰، واگن دوم شماره ۱، ... واگن پنجم شماره ۴. هر واگن می‌تواند یک مسافر (مقدار) داشته باشد. لیست هم دقیقاً همین است! یک ردیف از خانه‌های شماره‌گذاری شده.

✍️ ساخت لیست

لیست را با براکت [ ] می‌سازیم و آیتم‌ها را با کاما جدا می‌کنیم:

# لیست خالی empty = [] # لیست اعداد numbers = [10, 20, 30, 40] # لیست رشته‌ها names = ["Ali", "Sara", "Reza"] # لیست ترکیبی mixed = [1, "Hello", 3.14, True]

🔢 دسترسی به آیتم‌ها با ایندکس

هر آیتم در لیست یک شماره (ایندکس) دارد که از ۰ شروع می‌شود. اولین آیتم ایندکس ۰، دومین آیتم ایندکس ۱ و...

fruits = ["سیب", "موز", "پرتقال", "انگور"] print(fruits[0]) # سیب print(fruits[2]) # پرتقال # ایندکس منفی: از آخر print(fruits[-1]) # انگور (آخرین) print(fruits[-2]) # پرتقال (یکی مانده به آخر)
سیب پرتقال انگور پرتقال

✏️ تغییر مقدار یک آیتم

برخلاف رشته‌ها، لیست‌ها قابل تغییر هستند:

colors = ["قرمز", "سبز", "آبی"] colors[1] = "زرد" print(colors)
['قرمز', 'زرد', 'آبی']

📏 طول لیست با len()

nums = [5, 10, 15, 20] print(len(nums))
4

➕ اضافه کردن به لیست

items = ["A", "B"] items.append("C") # به انتها اضافه کن items.insert(1, "X") # در ایندکس ۱ قرار بده print(items)
['A', 'X', 'B', 'C']

➖ حذف از لیست

data = [10, 20, 30, 40] data.remove(30) # حذف مقدار 30 data.pop(1) # حذف ایندکس ۱ print(data)
[10, 40]
