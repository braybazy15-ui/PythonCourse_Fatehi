⌨️ گرفتن ورودی از کاربر (Input)

یاد می‌گیری چطور از کاربر اطلاعات بگیری و در برنامه استفاده کنی!

👋 سلام، دانشجو جان! امروز برنامه‌های تعاملی می‌سازیم ⌨️

⌨️ تابع input چیست و چه کاری انجام می‌دهد؟

تا حالا هر برنامه‌ای که نوشتیم، فقط خروجی داشت. یعنی ما به کامپیوتر چیزی می‌گفتیم و او چاپ می‌کرد. اما برنامه‌های واقعی تعاملی هستند! یعنی از کاربر ورودی می‌گیرند، روی آن پردازش انجام می‌دهند و نتیجه را نشان می‌دهند.

تابع input() دقیقاً همین کار را می‌کند: منتظر می‌ماند تا کاربر چیزی تایپ کند و دکمه Enter را بزند، سپس آن متن را به عنوان یک رشته (String) برمی‌گرداند.

📋

مثال واقعی: مثل فرم ثبت‌نام!

تصور کن یه فرم ثبت‌نام داری. اول از کاربر می‌پرسی: «اسمت چیه؟» کاربر اسمش رو می‌نویسه. بعد می‌پرسی: «چند سالته؟» کاربر سنش رو وارد می‌کنه. input() دقیقاً مثل این فرم‌ها از کاربر سوال می‌پرسه و جواب رو دریافت می‌کنه!

🎯 تعریف ساده: input() یعنی «از کاربر بپرس و جوابش رو بگیر». هر چیزی که کاربر تایپ کند، به صورت متن (رشته/String) دریافت می‌شود.

✍️ ساختار کلی تابع Input

# روش اول: گرفتن ورودی بدون پیام name = input() # روش دوم: گرفتن ورودی با پیام راهنما (بهتر!) name = input("لطفاً اسم خودت را وارد کن: ")

نکته: همیشه بهتر است داخل پرانتز input یک پیام راهنما بگذاری تا کاربر بداند چه چیزی باید وارد کند. در غیر این صورت، کاربر یک صفحه خالی می‌بیند و نمی‌داند باید چه کار کند!

💾 ذخیره ورودی در متغیر

خیلی مهم: ورودی که از کاربر می‌گیری را حتماً باید در یک متغیر ذخیره کنی، وگرنه از دست می‌رود و نمی‌توانی بعداً از آن استفاده کنی!

# ❌ غلط: ورودی را ذخیره نکردیم - از دست می‌رود! input("اسمت چیه؟ ") # ✅ درست: ورودی را در متغیر name ذخیره کردیم name = input("اسمت چیه؟ ") print("سلام", name)

🧪 مثال‌های ساده با Input

مثال ۱: سلام و احوالپرسی

name = input("لطفاً اسم خودت را وارد کن: ") print("سلام", name, "! خوش آمدی به دوره پایتون!")

نمونه اجرا:

لطفاً اسم خودت را وارد کن: Amir سلام Amir ! خوش آمدی به دوره پایتون!

مثال ۲: پرسیدن سن

name = input("اسمت چیه؟ ") age = input("چند سالته؟ ") print(f"اسم تو {name} و سن تو {age} سال است.")

⚠️ نکته خیلی مهم: ورودی همیشه رشته است!

این یکی از مهم‌ترین نکاتی است که باید بدانی: هر چیزی که با input دریافت می‌کنی، از نوع رشته (String) است، حتی اگر کاربر عدد وارد کند!

⚠️ اشتباه رایج:
age = input("سن تو: ") ← اگر کاربر عدد 20 را وارد کند، age برابر با "20" (رشته) است، نه 20 (عدد)!
پس نمی‌توانی با آن عملیات ریاضی انجام دهی مگر اینکه اول آن را به عدد تبدیل کنی!

🔄 تبدیل ورودی به عدد (Integer و Float)

برای اینکه بتوانی با عددی که کاربر وارد کرده، محاسبات ریاضی انجام دهی، باید آن را تبدیل کنی:

🔹 تبدیل به عدد صحیح (Integer)

# گرفتن سن و تبدیل به عدد age = input("سن تو چند ساله؟ ") age = int(age) # تبدیل رشته به عدد صحیح # حالا می‌تونیم محاسبه کنیم! next_year_age = age + 1 print(f"سال دیگه {next_year_age} ساله می‌شی!")

🔹 روش خلاصه‌تر (تبدیل همزمان با دریافت)

# دریافت و تبدیل همزمان در یک خط age = int(input("سن تو چند ساله؟ ")) print(f"سن تو دو برابر: {age * 2}")

🔹 تبدیل به عدد اعشاری (Float)

# گرفتن قد و تبدیل به عدد اعشاری height = float(input("قد تو چند متر است؟ ")) print(f"قد تو {height} متر است.")

⚠️ خطاهای رایج در کار با Input

نام خطاعلتراه‌حل
TypeError جمع عدد و رشته (فراموش کردن تبدیل) ورودی را با int() یا float() تبدیل کن
ValueError وارد کردن متن به جای عدد موقع تبدیل مطمئن شو کاربر واقعاً عدد وارد می‌کند
NameError ذخیره نکردن ورودی در متغیر همیشه ورودی را در یک متغیر بریز
EOFError فشردن Ctrl+D یا Ctrl+Z به جای وارد کردن متن ورودی را به درستی وارد کن

🎯 کاربردهای خلاقانه Input

  • 📝 فرم‌های ثبت‌نام: دریافت اسم، ایمیل، رمز عبور
  • 🧮 ماشین حساب: دریافت دو عدد و انجام عملیات روی آن‌ها
  • 🎮 بازی حدس عدد: کاربر یک عدد حدس می‌زند
  • 📊 محاسبه معدل: دریافت نمرات و محاسبه میانگین
  • 🛒 محاسبه قیمت: دریافت تعداد و قیمت واحد، محاسبه کل

🧪 تمرین‌های عملی Input

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب را چک کن!

۱
اسم کاربر را بگیر و به او سلام کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
name = input("اسمت چیه؟ ") print("سلام", name, "جان!")
۲
سن کاربر را بگیر و بگو سال دیگر چند ساله می‌شود
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
age = int(input("چند سالته؟ ")) print(f"سال دیگه {age + 1} ساله می‌شی!")
۳
دو عدد بگیر و مجموعشان را چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a = int(input("عدد اول: ")) b = int(input("عدد دوم: ")) print("مجموع:", a + b)
۴
طول و عرض مستطیل را بگیر و مساحت را چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
length = float(input("طول: ")) width = float(input("عرض: ")) area = length * width print("مساحت:", area)
۵
اسم و شهر کاربر را بگیر و در یک جمله چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
name = input("اسمت: ") city = input("شهرت: ") print(f"{name} اهل {city} است.")
۶
قیمت و تعداد را بگیر و مبلغ کل را حساب کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
price = float(input("قیمت: ")) count = int(input("تعداد: ")) total = price * count print("مبلغ کل:", total)
