مرور کامل input، کامنت‌ها، انواع داده و تبدیل آن‌ها

👋 سلام، دانشجو جان! امروز همه چیز را دوره می‌کنیم 📝

🎯 چرا مرور و جمع‌بندی مهم است؟

از مرحله ۹ تا ۱۲، چهار مبحث خیلی مهم را یاد گرفتی: گرفتن ورودی با input، کامنت‌گذاری، انواع داده (Data Types) و تبدیل انواع داده (Type Casting). این چهار مفهوم، پایه‌های اصلی برنامه‌نویسی هستند و قبل از ورود به مباحث پیشرفته‌تر مثل شرط‌ها، باید کاملاً بر آن‌ها مسلط باشی.

🧠 روش مطالعه این جلسه:
۱. خلاصه‌های هر درس را با دقت بخوان
۲. کدهای نمونه را ببین و سعی کن خودت بازنویسی کنی
۳. تمرین‌ها را حتماً خودت حل کن (بدون نگاه کردن به جواب!)
۴. روی کارت کلیک کن و جواب را چک کن
۵. در انتها، تست‌ها را بزن

📋 خلاصه درس Input

⌨️ تابع input()

از کاربر ورودی می‌گیرد. name = input("اسمت: ")

📝 همیشه رشته برمی‌گرداند

حتی اگر عدد وارد کنی، خروجی input رشته است!

🔄 تبدیل با int() و float()

برای محاسبات ریاضی: age = int(input("سن: "))

💾 حتماً در متغیر ذخیره کن

اگر ذخیره نکنی، ورودی از دست می‌رود!

📋 خلاصه درس کامنت‌ها

💬 کامنت تک‌خطی

با # شروع می‌شود: # این یک کامنت است

📄 کامنت چندخطی

با """ یا ''' محصور می‌شود.

🎯 هدف را توضیح بده

کامنت خوب می‌گوید «چرا»، نه «چه کاری»!

⌨️ کلید میانبر

در VS Code: Ctrl + /

📋 خلاصه درس انواع داده

📝 str (رشته)

متن: "Hello"

🔢 int (عدد صحیح)

اعداد بدون اعشار: 42, -10

📏 float (اعشاری)

اعداد با ممیز: 3.14, 5.0

✅ bool (بولین)

فقط True یا False

🔍 type()

تشخیص نوع داده: type(42) → int

📋 خلاصه درس تبدیل انواع داده

🔢 int()

تبدیل به عدد صحیح: int("42") → 42

📏 float()

تبدیل به اعشاری: float("3.14") → 3.14

📝 str()

تبدیل به رشته: str(42) → "42"

✅ bool()

تبدیل به بولین: bool(0) → False

🧪 ۱۲ تمرین متنوع

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب را چک کن!

۱
اسم کاربر را بگیر و به او سلام کن
👆 کلیک کن
✅ پاسخ:
name = input("اسمت چیه؟ ") print("سلام", name)
۲
دو عدد بگیر و مجموع را چاپ کن
👆 کلیک کن
✅ پاسخ:
a = int(input("اول: ")) b = int(input("دوم: ")) print("جمع:", a + b)
۳
سن کاربر را بگیر و سال تولد را حساب کن
👆 کلیک کن
✅ پاسخ:
age = int(input("سن: ")) year = 1403 - age print(f"سال تولد: {year}")
۴
نوع داده‌های مختلف را با type() نشان بده
👆 کلیک کن
✅ پاسخ:
print(type(42)) print(type("Hi")) print(type(3.14)) print(type(True))
۵
رشته "99" را به عدد تبدیل و دو برابرش کن
👆 کلیک کن
✅ پاسخ:
x = int("99") print(x * 2)
۶
عدد 50 را به رشته تبدیل و با متن ترکیب کن
👆 کلیک کن
✅ پاسخ:
score = 50 text = "امتیاز: " + str(score) print(text)
۷
برنامه‌ای با کامنت برای محاسبه مساحت دایره بنویس
👆 کلیک کن
✅ پاسخ:
# دریافت شعاع r = float(input("شعاع: ")) # محاسبه مساحت area = 3.14 * r * r print(f"مساحت: {area}")
۸
وزن و قد بگیر و BMI محاسبه کن
👆 کلیک کن
✅ پاسخ:
w = float(input("وزن: ")) h = float(input("قد: ")) bmi = w / (h ** 2) print(f"BMI: {bmi:.2f}")
۹
دمای سلسیوس را بگیر و به فارنهایت تبدیل کن
👆 کلیک کن
✅ پاسخ:
c = float(input("سلسیوس: ")) f = c * 9/5 + 32 print(f"فارنهایت: {f}")
۱۰
ساعت و دقیقه بگیر و به دقیقه کل تبدیل کن
👆 کلیک کن
✅ پاسخ:
h = int(input("ساعت: ")) m = int(input("دقیقه: ")) total = h * 60 + m print(f"{total} دقیقه")
۱۱
رشته "" و 0 را به bool تبدیل کن
👆 کلیک کن
✅ پاسخ:
print(bool("")) print(bool(0)) print(bool("Hi"))
۱۲
عدد 7.89 را به int تبدیل کن
👆 کلیک کن
✅ پاسخ:
x = int(7.89) print(x) # 7
