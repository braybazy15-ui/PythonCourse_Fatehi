🖨️ دستور Print - اولین برنامه پایتونی

سلام دنیا! شروع رسمی برنامه‌نویسی با پایتون

👋 سلام، دانشجو جان! امروز اولین کد واقعی رو می‌نویسیم 🖨️

🖨️ دستور Print چیست و چه کاری انجام می‌دهد؟

print() اولین و پرکاربردترین دستور در پایتون است. کار آن خیلی ساده است: هر چیزی که داخل پرانتز بهش بدی را روی صفحه نمایش (کنسول) چاپ می‌کند!

print یعنی «چاپ کن». البته چاپ روی کاغذ نیست! چاپ روی صفحه مانیتور، در همون پنجره‌ای که پایتون رو اجرا می‌کنی (CMD، Terminal یا Output در VS Code).

🎯 کاربرد اصلی: print راه ارتباط برنامه با تو است! وقتی می‌خواهی نتیجه یک محاسبه را ببینی، یک پیام به کاربر نشون بدی، یا بفهمی داخل برنامه چه خبر است، از print استفاده می‌کنی.

✍️ ساختار کلی دستور Print

print("متن مورد نظر") print(123) print(متغیر)

print یک تابع (Function) است. در پایتون، توابع را با گذاشتن پرانتز () بعد از اسمشان صدا می‌زنیم. هر چیزی داخل پرانتز بگذاریم، به تابع داده می‌شود تا روی آن کاری انجام دهد.

📝 چاپ متن (String) با Print

برای چاپ یک متن (رشته/String)، باید متن را داخل علامت کوتیشن (گیومه) قرار بدهیم. پایتون باید بفهمد که این یک متن است، نه اسم یک متغیر یا دستور خاص!

✅ انواع کوتیشن مجاز در پایتون:

  • دابل کوتیشن: "سلام دنیا" ← رایج‌ترین روش
  • سینگل کوتیشن: 'سلام دنیا' ← کاملاً معادل دابل کوتیشن
  • سه‌تایی دابل: """متن چند خطی""" ← برای متن‌های طولانی
  • سه‌تایی سینگل: '''متن چند خطی''' ← معادل سه‌تایی دابل
# مثال ۱: چاپ یک سلام ساده print("سلام به همه دوستان!") # خروجی: سلام به همه دوستان! # مثال ۲: با سینگل کوتیشن print('من عاشق پایتون هستم') # خروجی: من عاشق پایتون هستم # مثال ۳: متن چند خطی print("""خط اول خط دوم خط سوم""")

🔢 چاپ اعداد (Numbers) با Print

برای چاپ اعداد، نیازی به کوتیشن نیست! اعداد را مستقیماً داخل پرانتز می‌نویسیم. پایتون خودش می‌فهمد که این یک عدد است.

# چاپ عدد صحیح (Integer) print(42) # خروجی: 42 # چاپ عدد اعشاری (Float) print(3.14) # خروجی: 3.14 # چاپ عدد منفی print(-10) # خروجی: -10 # چاپ نتیجه یک محاسبه (نیاز به کوتیشن نیست!) print(5 + 3) # خروجی: 8 print(10 * 2) # خروجی: 20 print(100 - 45) # خروجی: 55
🧠 نکته طلایی: پایتون هر چیزی که داخل پرانتز باشد را اول محاسبه می‌کند، سپس نتیجه را چاپ می‌کند. پس print(5 + 3) اول 5+3 را حساب می‌کند (میشه ۸)، بعد ۸ را چاپ می‌کند!

🤔 اگر عدد را داخل کوتیشن بگذاریم چه می‌شود؟

این سوال خیلی مهمی است! ببینیم تفاوت کجاست:

# عدد بدون کوتیشن (عدد واقعی) print(123) # خروجی: 123 (به عنوان عدد) # عدد با کوتیشن (رشته متنی) print("123") # خروجی: 123 (به عنوان متن)

هردو ظاهراً یکسان چاپ می‌شوند، اما ماهیتشان فرق دارد:

  • 123 یک عدد است. می‌توانی باهاش جمع و تفریق کنی.
  • "123" یک رشته (متن) است. نمی‌توانی باهاش عملیات ریاضی انجام دهی!
# این کار می‌کنه (جمع دو عدد) print(100 + 23) # خروجی: 123 # این کار نمی‌کنه! (جمع عدد و رشته) print(100 + "23") # خطا! TypeError: unsupported operand type(s)

➕ ترکیب متن و عدد در Print

چند روش برای ترکیب متن و عدد داریم:

روش ۱: استفاده از کاما (ویرگول)

print("سن من:", 25) # خروجی: سن من: 25 print("قیمت:", 15000, "تومان") # خروجی: قیمت: 15000 تومان

کاما به صورت خودکار یک فاصله (Space) بین مقادیر قرار می‌دهد.

روش ۲: تبدیل عدد به رشته با str()

print("سن من: " + str(25)) # خروجی: سن من: 25

روش ۳: f-string (بهترین روش - در مراحل بعدی یاد می‌گیری)

name = "امیر" age = 20 print(f"سلام {name}، سن تو {age} سال است") # خروجی: سلام امیر، سن تو 20 سال است

📝 پارامترهای مهم تابع Print

تابع print پارامترهای مختلفی دارد که رفتار آن را کنترل می‌کنند:

🔹 پارامتر sep (Separator - جداکننده)

وقتی چند چیز را با کاما از هم جدا می‌کنی، بینشان فاصله می‌گذارد. با sep می‌تونی این جداکننده را عوض کنی!

print("الف", "ب", "ج") # خروجی: الف ب ج (فاصله پیش‌فرض) print("الف", "ب", "ج", sep="-") # خروجی: الف-ب-ج print("الف", "ب", "ج", sep=" ---> ") # خروجی: الف ---> ب ---> ج print("الف", "ب", "ج", sep="") # خروجی: الفبج (بدون فاصله)

🔹 پارامتر end (پایان خط)

print به طور پیش‌فرض بعد از چاپ، به خط بعدی می‌رود. با end می‌تونی این رفتار را عوض کنی!

# حالت پیش‌فرض: هر print در یک خط جدا print("سلام") print("دنیا") # خروجی: # سلام # دنیا # با end: هر دو در یک خط print("سلام", end=" ") print("دنیا") # خروجی: سلام دنیا # با end سفارشی print("مرحله ۱", end=" ---> ") print("مرحله ۲") # خروجی: مرحله ۱ ---> مرحله ۲

⚠️ خطاهای رایج در Print و راه‌حل آنها

نام خطاعلتراه‌حل
SyntaxError فراموش کردن پرانتز یا کوتیشن مطمئن شو پرانتز باز و بسته داری و متن داخل کوتیشن است
NameError چاپ یک متغیر تعریف‌نشده یا متن بدون کوتیشن متن را داخل کوتیشن بگذار یا متغیر را قبلاً تعریف کن
TypeError جمع عدد و رشته با + از کاما استفاده کن یا عدد را با str() به رشته تبدیل کن
SyntaxError مخلوط کردن سینگل و دابل کوتیشن اشتباه اگه متن شامل دابل کوتیشن است، از سینگل کوتیشن برای محصور کردن استفاده کن و برعکس
IndentationError فاصله یا تورفتگی اضافی قبل از print print باید از ابتدای خط شروع شود (بدون فاصله اضافی)
⚠️ اشتباه خیلی رایج: نوشتن print با حروف بزرگ! Print("سلام") ❌ غلط است! پایتون به بزرگ و کوچک بودن حروف حساس است. همیشه بنویس print با حروف کوچک.

🎯 کاربردهای خلاقانه Print

با print می‌تونی کارهای جالب و خلاقانه‌ای انجام بدی:

  • 🎨 رسم اشکال ساده: با استفاده از * و فاصله، مربع، مثلث و لوزی بکش!
  • 📊 نمودار متنی: با تکرار کاراکترها، نمودارهای ساده بساز.
  • 🎮 بازی‌های متنی: بازی حدس عدد، سنگ-کاغذ-قیچی، داستان تعاملی.
  • 📋 جدول و فرم: با sep و end می‌تونی جدول‌های مرتب بسازی.
# رسم یک مثلث ساده print(" * ") print(" *** ") print(" ***** ") print("*******")

🧪 تمرین‌های عملی Print

روی هر کارت کلیک کن تا پاسخ را ببینی! (حواست باشه: اول خودت حل کن، بعد جواب رو چک کن!)

۱
اسم خودت رو توی یه جمله چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("اسم من امیر است")
۲
حاصل جمع ۱۵ و ۳۲ رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print(15 + 32) # خروجی: 47
۳
اسم و سنت رو در یک خط چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("امیر", 20, "ساله") # خروجی: امیر 20 ساله
۴
سه کلمه رو با خط تیره (-) از هم جدا کن و چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("پایتون", "جاوا", "سی", sep="-") # خروجی: پایتون-جاوا-سی
۵
دو تا print پشت سر هم بنویس که خروجیشون توی یه خط باشه
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("سلام", end=" ") print("دنیا") # خروجی: سلام دنیا
۶
با print یه مربع ۳×۳ از ستاره بکش
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("***") print("***") print("***")
