مرور کامل print و متغیرها + ۱۰ تمرین متنوع

👋 سلام، دانشجو جان! امروز همه چیز رو دوره می‌کنیم 📝

🎯 چرا مرور و جمع‌بندی مهمه؟

تا اینجا دو تا از مهم‌ترین مفاهیم برنامه‌نویسی رو یاد گرفتی: print و متغیرها. حالا وقتشه که همه چی رو یه جا دوره کنیم، تمرین بیشتری حل کنیم، و مطمئن بشیم همه چی رو کامل یاد گرفتی قبل از اینکه بریم سراغ مباحث جدید!

🧠 روش مطالعه این جلسه: اول خلاصه‌ها رو بخون، بعد تمرین‌ها رو حتماً خودت حل کن (بدون نگاه کردن به جواب!)، سپس جواب رو چک کن. در آخر تست‌ها رو بزن.

📋 خلاصه درس Print

🖨️ تعریف Print

تابعی برای نمایش متن، عدد یا متغیر روی صفحه. هر چیزی داخل پرانتز بذاری رو چاپ می‌کنه.

📝 چاپ متن

متن باید داخل " " یا ' ' باشه: print("Hello")

🔢 چاپ عدد

عدد بدون کوتیشن: print(42) - می‌تونی محاسبات انجام بدی: print(5 + 3)

➕ ترکیب با کاما

با کاما چند چیز رو ترکیب کن: print("Age:", 25) ← فاصله خودکار

🔗 پارامتر sep

جداکننده بین مقادیر: print("a", "b", sep="-") ← a-b

⏭️ پارامتر end

پایان خط: print("Hi", end=" ") ← خط بعدی نمی‌ره

📋 خلاصه درس متغیرها

📦 تعریف متغیر

یه اسم برای ذخیره یه مقدار: name = "Ali"

📝 قوانین نام‌گذاری

با عدد شروع نشه، فاصله نداشته باشه، کاراکتر خاص ممنوع، فقط حروف و اعداد و _

🔄 تغییر مقدار

مقدار جدید جایگزین قدیمی میشه: x = 5 بعد x = 10

🧪 انواع داده

String (متن), Integer (عدد صحیح), Float (اعشاری), Boolean (True/False)

🖨️ با Print

با کاما یا f-string: print(f"Name: {name}")

⚠️ خطاها

NameError (تعریف نشده)، SyntaxError (اسم غیرمجاز)، TypeError (جمع عدد و رشته با +)

🧪 ۱۰ تمرین متنوع (Print + متغیر)

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب رو چک کن!

۱
اسم، سن و شهرت رو در متغیر بریز و با f-string در یک جمله چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
name = "Amir" age = 25 city = "Shiraz" print(f"I am {name}, {age} years old from {city}")
۲
دو عدد رو جمع کن و نتیجه رو با توضیح چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
a = 45 b = 55 result = a + b print("Sum:", result) # Sum: 100
۳
با sep، سه کلمه رو با خط تیره جدا کن و چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("Loading", end="...") print("Done!") # Loading...Done!
۵
مقدار متغیر score رو از 0 به 100 تغییر بده و هر مرحله چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
score = 0 print(score) # 0 score = 50 print(score) # 50 score = 100 print(score) # 100
۶
طول، عرض و ارتفاع مکعب رو در متغیر بریز و حجم رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
length = 10 width = 5 height = 3 volume = length * width * height print("Volume:", volume) # Volume: 150
۷
یه متغیر Boolean بساز و با f-string وضعیت دانشجو بودن رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
is_student = True print(f"Student status: {is_student}") # Student status: True
۸
سه متغیر رو در یک خط تعریف کن و هر سه رو چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
x, y, z = 10, 20, 30 print(x, y, z) # 10 20 30
۹
مقادیر دو متغیر color1 و color2 رو با هم عوض کن و چاپ کن
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
color1 = "Red" color2 = "Blue" color1, color2 = color2, color1 print(color1, color2) # Blue Red
۱۰
با print یه شکل مربع ۴×۴ از کاراکتر # بکش
👆 کلیک کن تا جواب رو ببینی
✅ پاسخ:
print("####") print("####") print("####") print("####")
