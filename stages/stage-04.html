🐍 پایتون چیه و چرا باید نصبش کنیم؟

پایتون یه زبان برنامه‌نویسی است که کدهایی که می‌نویسی رو خط به خط می‌خونه و اجرا می‌کنه. بهش می‌گن مفسر (Interpreter). برای اینکه بتونی کد پایتون بنویسی و اجرا کنی، اول باید خود پایتون رو روی کامپیوترت نصب کنی. این مثل این می‌مونه که بخوای با کسی تلفنی حرف بزنی، اول باید گوشی تلفن داشته باشی!

✅ خبر خوب: پایتون کاملاً رایگانه و روی ویندوز، مک و لینوکس کار می‌کنه. نصبش هم خیلی ساده‌ست و کمتر از ۵ دقیقه طول می‌کشه!

📥 دانلود پایتون

برای دانلود پایتون، باید بری به سایت رسمی پایتون:

🌐 آدرس سایت: https://www.python.org/downloads/
وارد این سایت بشو. خود سایت معمولاً آخرین نسخه پایتون رو بهت پیشنهاد میده. روی دکمه زرد رنگی که نوشته "Download Python" کلیک کن.

💻 نصب پایتون روی سیستم‌عامل‌های مختلف

🪟 ویندوز
🍎 مک
🐧 لینوکس

🪟 نصب روی ویندوز

  1. فایل دانلود شده (معمولاً با اسم python-3.x.x-amd64.exe) رو باز کن.
  2. خیلی مهم: حتماً تیک گزینه «Add Python to PATH» رو بزن! این کار باعث میشه از هر جای ویندوز بتونی پایتون رو اجرا کنی.
  3. روی «Install Now» کلیک کن و صبر کن تا نصب تموم بشه.
  4. بعد از اتمام نصب، پنجره Command Prompt رو باز کن (کلید Windows + R رو بزن، بنویس cmd و Enter کن).
  5. توی CMD بنویس: python --version و Enter بزن. اگه نسخه پایتون رو نشون داد (مثلاً Python 3.12.0)، یعنی نصب موفق بوده!
⚠️ نکته خیلی مهم: اگه تیک «Add Python to PATH» رو نزنی، موقع اجرای پایتون به خطا می‌خوری! اگه یادت رفت، باید پایتون رو دوباره نصب کنی یا مسیر پایتون رو دستی به PATH ویندوز اضافه کنی.

🍎 نصب روی مک

  1. فایل دانلود شده (با پسوند .pkg) رو باز کن.
  2. دستورالعمل‌های نصب رو دنبال کن (Continue > Continue > Agree > Install).
  3. بعد از نصب، برنامه Terminal رو باز کن (از Launchpad یا Spotlight جستجو کن).
  4. توی Terminal بنویس: python3 --version و Enter بزن. باید نسخه پایتون رو نشون بده.
🍎 نکته برای مک: مک معمولاً یه نسخه قدیمی از پایتون رو از قبل نصب داره (پایتون ۲). ما به پایتون ۳ نیاز داریم. برای اجرای پایتون ۳، همیشه از دستور python3 استفاده کن (نه python).

🐧 نصب روی لینوکس (اوبونتو)

  1. Terminal رو باز کن (Ctrl + Alt + T).
  2. اول مخازن رو آپدیت کن: sudo apt update
  3. بعد پایتون رو نصب کن: sudo apt install python3
  4. برای نصب pip (مدیریت بسته‌ها): sudo apt install python3-pip
  5. با دستور python3 --version نصب رو تأیید کن.
🐧 نکته: خیلی از توزیع‌های لینوکس از قبل پایتون ۳ رو نصب دارن. با دستور python3 --version می‌تونی چک کنی.

🧪 تست نهایی: مطمئن شو پایتون کار می‌کنه!

حالا که پایتون رو نصب کردی، بیا یه تست ساده انجام بدیم. توی CMD (ویندوز) یا Terminal (مک/لینوکس) این دستور رو بزن:

python (ویندوز) یا python3 (مک/لینوکس)

اگه یه چیزی شبیه این دیدی، یعنی همه چی عالیه:

Python 3.12.0 (main, Oct 2 2023, 13:22:15)
Type "help", "copyright", "credits" or "license" for more information.
>>>

اون علامت >>> یعنی پایتون آماده‌ست و منتظر دستورات توئه! برای خروج، بنویس exit() و Enter بزن.

🎉 تبریک! پایتون با موفقیت روی سیستم تو نصب شد! حالا آماده‌ای که بریم سراغ نصب VS Code (مرحله بعد) و بعدش اولین کد پایتون رو بنویسیم! 🚀
