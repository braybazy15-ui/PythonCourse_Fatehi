📝 مرور و جمع‌بندی ۱

مرور کامل دستور print و متغیرها + ۱۲ تمرین متنوع

👋 سلام، دانشجو جان! امروز همه چیز رو دوره می‌کنیم 📝

🎯 چرا مرور و جمع‌بندی مهم است؟

تا اینجای کار، دو تا از مهم‌ترین و پایه‌ای‌ترین مفاهیم برنامه‌نویسی رو یاد گرفتی: دستور print برای نمایش خروجی و متغیرها برای ذخیره‌سازی اطلاعات. این دو مفهوم، الفبای برنامه‌نویسی هستند. بدون تسلط بر این‌ها، یادگیری مباحث بعدی سخت می‌شود.

حالا وقتش رسیده که همه چیز رو یک‌جا دوره کنیم، تمرین‌های بیشتری حل کنیم، و مطمئن بشویم که پایه‌ات کاملاً محکم شده است. یادت باشد: برنامه‌نویسی با دیدن و خواندن یاد گرفته نمی‌شود، با نوشتن و تمرین کردن یاد گرفته می‌شود!

🧠 روش مطالعه این جلسه:
۱. ابتدا خلاصه‌های هر درس را با دقت بخوان
۲. تمرین‌ها را حتماً خودت حل کن (بدون نگاه کردن به جواب!)
۳. روی کارت کلیک کن و جواب را ببین
۴. اگر اشتباه کردی، دوباره تمرین را حل کن
۵. در انتها، تست‌ها را بزن تا سطح خودت را بسنجی

📋 خلاصه کامل درس Print

🖨️ تعریف Print

تابعی برای نمایش متن، عدد یا متغیر روی صفحه نمایش. هر چیزی داخل پرانتز بگذاری را چاپ می‌کند.

📝 چاپ متن (رشته)

متن باید داخل " " یا ' ' باشد: print("Hello")

🔢 چاپ عدد

عدد بدون کوتیشن نوشته می‌شود: print(42) - می‌توانی محاسبات ریاضی انجام دهی: print(5 + 3)

➕ ترکیب با کاما

با کاما چند مقدار را ترکیب کن: print("سن:", 25) ← به‌طور خودکار فاصله می‌گذارد

🔗 پارامتر sep

جداکننده بین مقادیر را تعیین می‌کند: print("الف", "ب", sep="-") ← الف-ب

⏭️ پارامتر end

پایان خط را تعیین می‌کند: print("سلام", end=" ") ← به خط بعدی نمی‌رود

📋 خلاصه کامل درس متغیرها

📦 تعریف متغیر

یک اسم برای ذخیره یک مقدار: name = "Ali"

📝 قوانین نام‌گذاری

با عدد شروع نشود، فاصله نداشته باشد، کاراکتر خاص ممنوع، فقط حروف انگلیسی و اعداد و _

🔄 تغییر مقدار

مقدار جدید جایگزین قدیمی می‌شود: x = 5 سپس x = 10

🧪 انواع داده

String (متن), Integer (عدد صحیح), Float (اعشاری), Boolean (درست/نادرست)

🖨️ استفاده با Print

با کاما یا f-string: print(f"نام: {name}")

⚠️ خطاهای رایج

NameError (تعریف نشده)، SyntaxError (اسم غیرمجاز)، TypeError (جمع عدد و رشته با +)

🧪 ۱۲ تمرین متنوع (Print + متغیر)

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن، بعد جواب را چک کن!

۱
اسم خودت را در یک متغیر بریز و چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
name = "Amir" print(name) # خروجی: Amir
۲
دو عدد را جمع کن و نتیجه را با توضیح چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a = 45 b = 55 print("مجموع:", a + b) # خروجی: مجموع: 100
۳
با sep سه کلمه را با خط تیره جدا کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
print("Python", "Java", "C++", sep=" - ") # خروجی: Python - Java - C++
۴
با end دو print را در یک خط بنویس
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
print("در حال بارگذاری", end="...") print("تمام شد!") # خروجی: در حال بارگذاری...تمام شد!
۵
مقدار score را از 0 به 100 تغییر بده و چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
score = 0 print(score) # 0 score = 100 print(score) # 100
۶
طول و عرض مستطیل را در متغیر بریز و مساحت را چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
length = 10 width = 5 area = length * width print("مساحت:", area) # خروجی: مساحت: 50
۷
با f-string اسم و سن را در یک جمله چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
name = "Sara" age = 22 print(f"اسم من {name} و سن من {age} است") # خروجی: اسم من Sara و سن من 22 است
۸
سه متغیر را در یک خط تعریف کن و چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
x, y, z = 10, 20, 30 print(x, y, z) # خروجی: 10 20 30
۹
مقادیر دو متغیر را با هم عوض کن و چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
first = "قرمز" second = "آبی" first, second = second, first print(first, second) # خروجی: آبی قرمز
۱۰
یه متغیر Boolean بساز و وضعیت را چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
is_student = True print(f"وضعیت دانشجویی: {is_student}") # خروجی: وضعیت دانشجویی: True
۱۱
حاصل ضرب سه عدد را در متغیر بریز و چاپ کن
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
a, b, c = 2, 3, 4 result = a * b * c print("حاصل ضرب:", result) # خروجی: حاصل ضرب: 24
۱۲
با print یک مثلث ساده با ستاره بکش
👆 کلیک کن تا جواب را ببینی
✅ پاسخ:
print(" * ") print(" *** ") print("*****") # خروجی: # * # *** # *****
