🔄 حلقه while در پایتون

تکرار تا وقتی که شرط برقرار باشد!

👋 سلام، دانشجو جان! امروز حلقه while را یاد می‌گیریم 🔄

🔄 حلقه while چیست و چه فرقی با for دارد؟

حلقه while برخلاف for که برای تعداد مشخص استفاده می‌شود، برای وقتی است که نمیدانی چند بار باید کد تکرار شود! while یعنی «تا وقتی که». تا وقتی که شرطی برقرار است، کد داخل حلقه را اجرا کن.

تفاوت اصلی:
• for: می‌دانی چند بار (مثلاً ۱۰ بار، یا به اندازه یک لیست)
• while: نمی‌دانی چند بار (تا وقتی که کاربر حدس درست بزند، تا وقتی که به جواب برسی)

🧠 مثال واقعی: تصور کن داری با آب سطل، لیوان را پر می‌کنی. نمی‌دانی دقیقاً چند ثانیه طول می‌کشد. فقط تا وقتی که لیوان پر نشده، به ریختن آب ادامه می‌دهی. این دقیقاً حلقه while است! «تا وقتی لیوان پر نشده (شرط)، آب بریز (کد داخل حلقه)».

✍️ ساختار حلقه while

while شرط: # کدی که تا وقتی شرط درست است اجرا می‌شود print("هنوز شرط برقراره...")
⚠️ خطر حلقه بی‌نهایت! اگر شرط while هیچوقت False نشود، برنامه تا ابد اجرا می‌شود و کامپیوترت هنگ می‌کند! همیشه مطمئن شو که یک راه برای خارج شدن از حلقه وجود دارد. برای متوقف کردن برنامه در VS Code: Ctrl + C

📝 مثال‌های ساده با while

مثال ۱: شمارش ۱ تا ۵

i = 1 while i <= 5: print(i) i = i + 1
1 2 3 4 5

مثال ۲: شمارش معکوس

count = 5 while count > 0: print(count) count = count - 1 print("پرتاب!")
5 4 3 2 1 پرتاب!

🎮 بازی حدس عدد با while

یکی از بهترین کاربردهای while، بازی‌های تعاملی است:

secret = 7 guess = 0 while guess != secret: guess = int(input("حدس بزن: ")) if guess < secret: print("برو بالاتر!") elif guess > secret: print("بیا پایین‌تر!") print("🎉 آفرین! درست حدس زدی!")

🚪 خروج از حلقه با break

break یعنی «بس کن!». هر جا از break استفاده کنی، فوراً از حلقه خارج می‌شوی.

while True: name = input("اسمت رو بگو (یا quit برای خروج): ") if name == "quit": break print(f"سلام {name}!")

⏭️ رد کردن یک دور با continue

continue یعنی «این دور رو رد کن، برو دور بعد!». کدهای بعد از continue در آن دور اجرا نمی‌شوند.

i = 0 while i < 5: i = i + 1 if i == 3: continue print(i)
1 2 4 5

📊 مقایسه for و while

✅ از for استفاده کن وقتی:
• تعداد تکرار مشخص است
• می‌خواهی روی اعضای یک مجموعه (لیست، رشته، range) حرکت کنی

✅ از while استفاده کن وقتی:
• نمی‌دانی چند بار باید تکرار شود
• منتظر یک اتفاق خاص هستی (مثلاً ورودی کاربر)
• شرط خروج پیچیده‌ای داری

🧪 تمرین‌های عملی while

روی هر کارت کلیک کن تا پاسخ را ببینی! اول خودت حل کن!

۱
با while اعداد ۱ تا ۱۰ را چاپ کن
👆 کلیک کن
✅ پاسخ:
i = 1 while i <= 10: print(i) i = i + 1
۲
برنامه‌ای که تا نوشتن exit از کاربر ورودی بگیرد
👆 کلیک کن
✅ پاسخ:
while True: x = input("متن: ") if x == "exit": break print(x)
۳
جمع اعداد تا وقتی که کاربر ۰ وارد کند
👆 کلیک کن
✅ پاسخ:
t = 0 n = 1 while n != 0: n = int(input("عدد: ")) t = t + n print("جمع:", t)
۴
رمز عبور بگیر تا رمز صحیح وارد شود
👆 کلیک کن
✅ پاسخ:
p = "" while p != "1234": p = input("رمز: ") print("✅ ورود")
۵
چاپ اعداد زوج ۲ تا ۲۰ با while
👆 کلیک کن
✅ پاسخ:
i = 2 while i <= 20: print(i) i = i + 2
۶
شمارش تعداد ارقام یک عدد
👆 کلیک کن
✅ پاسخ:
n = int(input("عدد: ")) c = 0 while n > 0: n = n // 10 c = c + 1 print("تعداد ارقام:", c)
